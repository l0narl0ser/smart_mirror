# Smart Mirror Control

Приложение для управления умным зеркалом (Smart Mirror) через MQTT протокол.

## Функционал

### Подключение к MQTT
- Настройка подключения к MQTT брокеру (host, port, topic prefix)
- Отображение статуса подключения в реальном времени
- Автоматическая подписка на топики зеркала

### Управление будильником
- Установка будильника (часы и минуты)
- Очистка будильника
- Отображение текущего будильника

### Управление фразами
- Ввод пользовательских фраз для отображения на зеркале
- Очистка фразы
- Отображение текущей фразы

### Управление устройством
- Перезапуск устройства (с подтверждением)

### Статус и лог
- Отображение статуса подключения
- Лог последних полученных сообщений

## MQTT Топики

| Команда | Топик | Payload |
|---------|-------|---------|
| Установить будильник | `mirror/alarm/set` | `{"hour": 7, "minute": 30}` |
| Очистить будильник | `mirror/alarm/clear` | `{}` |
| Перезапустить устройство | `mirror/device/restart` | `{}` |

## Зависимости

- `mqtt_client` — MQTT клиент
- `shared_preferences` — локальное хранение настроек

## Запуск

```bash
flutter pub get
flutter run
```

## Настройка

1. Откройте настройки (иконка шестеренки)
2. Укажите адрес MQTT брокера (IP Raspberry Pi)
3. Укажите порт (по умолчанию 1883)
4. Нажмите Connect
